## 全量レビューを実行する

1. AWSコンソールに遷移
2. Amazon CodeGuruに遷移
3. ご利用開始にあたってのプルダウンから「CodeGuru Reviewer」を選択し、ご利用開始にあたってボタンを押下。
4. メニューから設定する。
    - リポジトリに下記の選択肢がある。
        - AWS CodeBuild
        - Bitbucket
        - Github or Github Enterprise Cloud
        - Github Enterprise Server
5. しばらく待つ
